Dhara — The Agent Protocol Standard

A protocol-first, language-agnostic AI coding agent harness.

Key differences from existing agent tools:
• Extensions communicate via JSON-RPC 2.0 wire protocol, not language-specific APIs
• Built-in capability-based security (declare → approve → enforce)
• 20+ LLM providers via single adapter (OpenAI, Anthropic, Google, Groq, etc.)
• Write extensions in ANY language (Python, Rust, shell script, TypeScript)

The 6 standard tools (read, write, edit, ls, grep, bash) are the minimum. Everything else is an extension — proving that network tools, git operations, code search, and test runners all belong in the extension layer.
